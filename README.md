Undefine website 2016
==================

Run
----

```console
$ # Install deps
$ npm install
$ # Do an initial build
$ gulp build
$ # Launch the dev server
$ gulp
```

Audio Sprites
----

Run following command to generate new audio sprite

```console
$ gulp audio
```

Requirements
```console
$ brew reinstall ffmpeg --with-theora --with-libogg --with-libvorbis
```
